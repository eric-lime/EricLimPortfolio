Company Database!

"The Office" broken down!

Within this project I was able to create a database based off of the TV show "The Office" with the help of FigJam, Excel, SQL, and MySQL.

Step-by-Step: In order to create a Database, I first had to create an ER diagram (Entity Relationship) so that I could have a data model to base my database schema off of. Next, I was able to take the diagram and convert it into a schema through Excel. Finally, I used the schema that I had made to develop the database with the help of a Relational Database Management System(MySQL) and SQL.

In order to create the database I had to use DML or Data Manipulation Language in order to adjust the data to make organised and easier to read. This included inserting, deleting, and modifying the data and entries in order to do so.

Finally, with a set database, I was able to then perform querying methods(DQL) in order to retrieve insight that was desired!

Overall, this project was very intriguing and helped me learn a lot about the processes it takes to develop a concept and turn it into an actually database which you can work with and grab insight from. I also had the opportunity to develop trigger tests for the database.

A blast!
